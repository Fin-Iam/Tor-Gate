## Notes
The application implements a 3-layer security gate.
Layer 1: DDOS check (simulated delay).
Layer 2: Onion URL Captcha (missing character challenge).
Layer 3: GPG-based 2FA Login/Registration.
Theme: Dark hacker terminal/cyber-security aesthetic.
